Python环境搭建

在开始学习Python编程之前，我们需要先搭建开发环境。本章节将指导您完成Python解释器的安装、开发环境的配置，并创建第一个Python程序。

下载Python

首先，我们需要从Python官方网站下载Python解释器。

Python版本选择

推荐下载Python 3.x版本（最新稳定版），因为Python 2.x已经停止维护。本教程使用Python 3.9作为示例。

下载步骤：

  1. 访问Python官网：https://www.python.org/downloads/
  2. 点击"Download Python 3.9.x"按钮
  3. 选择适合您操作系统的安装包（Windows/macOS/Linux）

安装Python

Windows安装步骤

  1. 双击下载的安装包（例如：python-3.9.x-amd64.exe）
  2. 勾选"Add Python 3.9 to PATH"选项
  3. 选择"Install Now"进行标准安装
  4. 等待安装完成

注意：勾选"Add Python to PATH"非常重要，这样可以在命令行中直接使用Python命令。

macOS安装步骤

  1. 双击下载的安装包（.pkg文件）
  2. 按照安装向导的提示进行安装
  3. 输入管理员密码授权安装
  4. 等待安装完成

Linux安装步骤

大多数Linux发行版已预装Python。如果需要安装，可以使用包管理器：

# Ubuntu/Debian
sudo apt update
sudo apt install python3

# CentOS/RHEL
sudo yum install python3

验证安装

安装完成后，我们需要验证Python是否正确安装：

  1. 打开命令行终端（Windows按Win+R，输入cmd；macOS打开终端）
  2. 输入以下命令：
python --version
# 或
python3 --version

如果看到类似以下输出，说明安装成功：

Python 3.9.x

Python安装时已包含

第一个Python程序

让我们创建第一个Python程序来验证环境是否正确配置：

  1. 创建一个新文件，命名为 hello.py
  2. 输入以下代码：
# 这是我的第一个Python程序
print("Hello, Python!")

# 输入你的名字
name = input("请输入你的名字：")
print(f"你好，{name}！欢迎学习Python！")

运行程序：

  1. 打开命令行终端
  2. 进入文件所在目录
  3. 输入命令：python hello.py

预期输出：

Hello, Python!
请输入你的名字：小明
你好，小明！欢迎学习Python！
